# Sales & Demand Forecasting - FUTURE_ML_01

## Objective
To forecast future retail sales using historical transaction data to support business decision-making.

## Project Overview
This project uses historical Superstore sales data to:

- Clean and prepare time-based data
- Perform feature engineering
- Aggregate monthly sales
- Build a regression-based forecasting model
- Predict the next 12 months of sales
- Visualize actual vs forecasted trends

## Tools Used
- Python
- Pandas
- NumPy
- Scikit-learn
- Matplotlib

## Business Impact
The forecast helps businesses:

- Plan inventory
- Manage staffing
- Improve budgeting
- Reduce overstocking risks

## Forecast Output
The model predicts future monthly sales trends based on historical data.
